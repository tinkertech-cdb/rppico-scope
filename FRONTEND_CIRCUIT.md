# FRONTEND_CIRCUIT

Analog frontend and function-generator output notes for the MVP hardware profile.

MVP board: Seeed XIAO RP2350

Portability target: RP2350-class boards

## 1) Channel Mapping

- Scope CH1: `ADC0`
- Scope CH2: `ADC1`
- Function generator output: `ADC2 pin` in GPIO/PWM mode (not ADC input mode)

## 2) Scope Input Frontend (MVP)

Planned per-channel conditioning:

- Passive voltage divider
- Mid-bias offset at `1.65V`
- Intended external signal envelope: `-1.6V..+1.6V` (`3.2Vpp`)

This maps bipolar input behavior into single-supply ADC range.

## 3) Design Goals

- Protect RP2350 ADC pins
- Preserve usable bandwidth for MVP scope modes
- Keep frontend simple and reproducible
- Keep CH1/CH2 behavior closely matched

## 4) Recommended Per-Channel Blocks

1. Input connector
2. Protection stage (series resistor and clamping strategy)
3. Attenuation divider
4. 1.65V bias injection/summing node
5. Optional RC anti-alias filter
6. ADC pin (`ADC0` or `ADC1`)

## 5) Bias Network Guidance

- Use a stable 1.65V reference node
- Prefer a buffered or otherwise low-impedance bias source if loading varies
- Decouple bias node near the analog path

## 6) Protection Guidance

Minimum practical protection per channel:

- Series input resistor to limit fault current
- Clamp path to keep ADC pin within safe range
- Grounding/layout that reduces digital-noise injection

## 7) Function Generator Output (PWM)

FG output is PWM-generated on `ADC2 pin` configured as GPIO output.

Recommended output chain:

1. PWM pin
2. RC low-pass filter (single or multi-stage, based on ripple target)
3. Optional output buffer/amplifier
4. Optional protection resistor at output connector

Notes:

- Sine/triangle/saw quality depends on PWM carrier, filter topology, and load.
- Square waveform can be emitted directly (or lightly conditioned).
- DC output is generated by PWM duty cycle plus filtering.

## 8) Practical Constraints

Guaranteed on-chip ADC envelope:

- 500 kS/s aggregate
- ~250 kS/s per channel in 2CH mode

## 9) PCB/Layout Checklist

- Keep analog traces short and away from fast digital and PWM edges
- Use controlled analog/digital return strategy
- Place protection/filter parts close to ADC pins and connectors
- Decouple supplies near analog nodes and RP2350 pins

## 10) Initial Validation

- Loop back FG to scope channel through known attenuation path
- Verify:
  - offset centering near expected mid-scale
  - clipping margins
  - frequency tracking
  - FFT peak placement consistency
- Measure baseline noise with grounded input on both channels

## 11) Next Revision Targets

- Switchable gain/attenuation ranges
- Improved offset reference implementation
- External ADC path for higher simultaneous-channel fidelity
